# 보평중 1-3 시간표 Flutter 앱 + 위젯 설계

## 개요

보평중학교 1학년 3반 시간표를 보여주는 안드로이드 앱 + 홈 화면 위젯.
기존 PWA 앱의 네이티브 버전으로, 홈 화면 위젯 지원이 핵심 기능.

## 핵심 기능

### 1. 앱 메인 화면
- 주간 시간표 테이블 표시
- 주차 선택 드롭다운
- 오늘 컬럼 하이라이트
- 현재 교시 강조
- 오프라인 캐시 지원

### 2. 홈 화면 위젯 (4x4)
- 풀 주간 시간표 표시
- 각 셀: 과목 (위) + 선생님 (아래) 2줄 구성
- 오늘 컬럼 배경색 하이라이트
- 현재 교시 테두리 강조
- 위젯 클릭 시 앱 열기
- 주기적 자동 갱신 (1시간마다)

### 3. API 통신
- 엔드포인트: `https://estelle-hub.mooo.com/api/comci/36179?{params}`
- 파라미터: base64(`73629_27224_0_{주차}`)
- 응답 처리: null 바이트 제거 후 JSON 파싱

### 스타일
- 폰트: 과목 11sp bold, 선생님 9sp regular
- 오늘 컬럼: #E3F2FD (연한 파랑)
- 현재 교시: #FFF9C4 (연한 노랑) + 테두리
- 일반 셀: #FFFFFF

## 데이터 흐름

1. 앱/위젯 시작
2. SharedPreferences에서 캐시 확인
3. API 호출 (캐시 없거나 오래된 경우)
4. 응답 파싱 (null 바이트 제거)
5. SharedPreferences에 저장
6. UI 갱신 (앱) / 위젯 업데이트 (home_widget)

## 코드 해석

컴시간 API 응답의 시간표 코드:
```
코드 = XXYYY (5자리)
- 과목코드 = XX (코드 ÷ 1000)
- 교사코드 = YYY (코드 % 1000)

예: 23033 → 과목 23(체육), 교사 33(윤재)
```
